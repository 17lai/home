> 注意事项

1. 自己定制运动量，建议**循序渐进，感到身心愉悦**，不要一次运动就搞得筋疲力尽，浑身酸痛
2. **不要运动过量，运动不合理，伤害身体**（好好研究你的姿势对不对）
3. 一定要设计休息恢复日。休息日不是什么都不做。**轻量热身运动，瑜伽，泡沫轴筋膜枪按摩恢复都可以做**（大保健伤不伤身体？）
4. 饮食是核心，**增肌需要热量盈余，减脂需要热量缺口**。蛋白不停的损失，一定要多吃一些容易吸收的动物蛋白类食物。
